这里是关于手机端的配置和安装，手机端的小狼毫输入法叫同文输入法，附上下载地址。安装完成之后并不能打出中文，还需要其它配置文件另外下载。一个是基础配置，还有就是其它输入法[1]（五笔画、双拼等等）以下是配置[2]：

基础配置文件

下载一个rimerc-trime-0.x.x.zip 压缩文件

  • 备份自己原有配置文件，如将原配置文件夹 rime 重命名为 rime.old
  • 解压刚刚下载的压缩包，把里面的trime文件夹命名为rime之后复制(/剪切)到原有配置文件的路径
  • 进入刚刚安装的同文输入法
    • 点击部署（部署的时间有点长）

输入法选择

  • ℞ luna-pinyin: 朙月拼音 / Pinyin input method for Tranditional Chinese

  • ℞ terra-pinyin: 地球拼音 / School-taught Pinyin, with tone marks

  • ℞ bopomofo: 注音 / Zhuyin (aka. Bopomofo)

  • ℞ pinyin-simp: 袖珍簡化字拼音

  • ℞ double-pinyin: 雙拼 / Double Pinyin (ZiRanMa, ABC, flyPY, MSPY, PYJJ variants)

  • ℞ combo-pinyin: 宮保拼音 / Combo Pinyin, a chord-typing input method

  • ℞ stenotype: 打字速記法 / a stenographic system derived from ABC Easy Shorthand

和上面一样，下载压缩包解压之后把后缀是.yaml放入rime下，然后打开安装的同文输入法点击部署等部署完之后点击选取软键盘选择同文输入法既可以使用了。

  1. 1.选择自己喜欢的输入法就可以不用全部下载部署 ↩
  2. 2.更换默认主题，选择键盘选择同文风就可以了 ↩

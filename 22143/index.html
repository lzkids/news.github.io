  • 2. HTML代码规范介绍
    1. 1.嵌套元素应当缩进一次（即两个空格）
    2. 2.对于属性的定义，确保全部使用双引号，不要使用单引号。
    3. 3.单标签中不要添加‘/’结束
    4. 4.不能省略双标签的结束的标签
    5. 5.在引入CSS文件和JS文件时候，不需要指定type属性
    6. 6.标签使用过程中，要遵循标签的语义化
    7. 7.标签设置属性的时候，要按照顺序依次设置，保证代码易读性
  • 3.CSS代码规范介绍
    1. 1. 为了代码的易读性，在每个声明块的左花括号前添加一个空格，且单独成行
    2. 2.每条声明语句的：后应该插入一个空格,且以封号结束
    3. 3.避免为 0 值指定单位
    4. 4. 声明顺序
    5. 5. 选择器
  • 4.js语法规范
    1. ☞定义变量后要空行,在定义变量的同时初始化该变量
    2. ☞函数名之后不要留空格
    3. ☞单目运算符 !、~、++、－－、－、*、& 等前后不加空格
    4. ☞声明数组使用[]
    5. ☞通常能用CSS实现的效果要避免使用JS来实现
    6. ☞变量要使用驼峰命名法，使用名词结构
    7. ☞构造函数使用帕斯卡命名法
    8. ☞对象属性命名必须唯一
    9. ☞在同一个作用域下，变量名和方法名不要重复
    10. ☞每一个方法之间空一行
    11. ☞运算符前后需要添加空格
    12. ☞紧挨着圆括号内不需要加空格
    13. ☞三目运算符用来进行条件性的赋值，不要用来当做简写的 if 语句使用。

Linux 是什么？

Linux 是一个开源操作系统和 IT 基础架构平台。它由 Linus Torvalds 于 1991 年构思设计而成，

Linux 的优点

跨平台的硬件支持

低资源占用下良好的稳定性

高效的文件系统

可靠的安全性

完善纯净的软件包管理机制

随心所欲定制 UI

社区的支持

开源

学习和探索

Linux 因为其完全模块化的基理，干净得像水一样。水本身无固定的形态，它可以随环境而作出适应，让自己变成外物的形态。

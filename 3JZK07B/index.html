在网页msys2下载安装；
完成后，勾选立即运行 MSYS2；
更新包数据库和基本包。除非您的安装文件是最新的，否则需要两个步骤。首次运行 ：
pacman -Syu
从“开始”菜单运行“MSYS2 MSYS”。使用 以下命令更新其余的基本包：
pacman -Su
现在 MSYS2 已为您准备就绪。您可能需要安装一些工具和 mingw-w64 GCC 来开始编译：
pacman -S --needed base-devel mingw-w64-x86_64-toolchain
打开Visual Studio Code

  • 按Ctrl+Shift+P运行C/C++:编辑配置(UI)打开后填写几个配置
    • 配置名称处
      • 删掉Win32填写GCC
  • 编译器路径
    • C:/msys64/mingw64/bin/g++.exe（下拉选择你的安装路径）
  • IntelliSense模式
    • 选择gcc-x64
